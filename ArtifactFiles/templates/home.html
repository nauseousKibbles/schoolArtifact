What software do Software Developers use?

Based On Job Listings...

Based on a job listing, the company Jobot requires Software Developers to know one or more of the following: Angular, JavaScript, TypeScript, HTML, CSS, C#, Java, C++.

The company Flexion, for a Remote Full Stack Engineer, requires you to have 8+ years of software engineering experience, know Java Spring, Python, NodeJS, Amazon States Language, Docker, AWS Lambda, and AWS Step Functions. For databased, they need you to know AWS PostgreSQL RDS, AWS Redshift, and S3. Those are all with Amazon Web Services.

A Full Stack Developer or Engineer specialized in front-end and back-end development. Front-end development is all of the stuff the user sees and interacts with. Back-end development is everything the user doesn't see, so databases, or the server hosting the website.

Other Software

From personal experience with programming and gaining information from searching the internet, many people use Git to store their projects. Git is a program that keeps track of changes to software you made so you can see who did what, and when. It allows many people to work on the same project without causing problems with interfering. There are websites that host Git servers. Github and Gitlab are two of them. The Flexion company requires you to know Gitlab. I personally use Github and I am keeping track of this website's code on it. A website that I see being used a lot of Stackoverflow.

What Do Software Developers Do?

Software Developers develop software to help a customer's needs. That could include: an internal system upgrade, updating to a new version of a dependency, or to create something new, like a website or the program on the McDonald's kiosks.

Software Developers can develop software for a lot of different areas of the computer field. They aren't limited to websites or applications.
